# 🌤️ Weather Application

A responsive weather application built with **HTML**, **CSS**, and **JavaScript** that provides real-time weather information for any city around the world.

---

## ✨ Features

- 🔍 Search weather by city name
- 🌡️ Displays current temperature, humidity.
- 🌥️ Shows weather condition with temperature
- 🎨 Clean and modern UI

---

## 🛠️ Technologies Used

- **HTML5** – Markup structure
- **CSS3** – Styling and responsive layout
- **JavaScript (ES6+)** – Logic and API integration
- **OpenWeatherMap API** – Real-time weather data
